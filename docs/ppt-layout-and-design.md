# PPT layout and design

How to build a PPT:

1. Write the title and content of every page in plain text.
2. Sort the content on each page: group information, cut filler, and split into hierarchy (overview, sub-points, with a heading for each part).
3. Pick the colour scheme.
4. Lay out each page.
5. Design the shape of every element on the page.
