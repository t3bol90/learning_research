# Experiment Notes Template

## Goal of the experiment

> Describe why you are running this experiment and what you want to learn from it.

## Experimental setting

> Which data the experiment runs on, and what was changed in the algorithm.

## Record the results

> Log both the results that worked and those that did not, including visualisations and quantitative numbers.

## Analyse the results, possible causes

> Check whether the outcome matches your expectations. If it does not, [work out why the experiment is not working](./why-experiment-isnt-working.md) (Notion).

## Experiments to run next

> You are the project lead. Keep thinking about what to do next, and list the experiments you plan to run rather than waiting for instructions.
